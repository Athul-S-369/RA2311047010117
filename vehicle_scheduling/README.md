# Vehicle scheduling (evaluation folder name)

Per the evaluation brief, submissions reference a **`vehicle_scheduling`** folder name.

The implemented microservice, scripts, and screenshots for this track live in **`vehicle_maintence_scheduler/`** at the repository root (same codebase you run with `npm start` there).

Place **vehicle** run screenshots under **`vehicle_maintence_scheduler/screenshots/`** (or copy them here if your zip layout requires this exact folder name).
